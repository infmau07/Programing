Conclusión

Durante esta segunda unidad, se realizaron tanto actividades de investigación, se vieron nuevos elementos llegando a ser aprendidos.

Varios de estos nuevos elementos a resaltarcon Net MAUI, conformado por Stack Layout, Absolute Layout, Grid y Flex Layout.

Estos cuatro fueron enseñados paso a paso de como utilizarlos, además de varios ejercicios y problemas que se tenian que resolver utilizandolos-

Algunos ejemplos de esos problemas realizados con Stack Layout, fue uno en el que de acuerdo a la fecha ingresada, este te daria tu edad actual, otro problema pero ahora con Absolute Layout fue uno donde nosotros debiamos ingresar tres numeros distintos y al momento de presionar el botón, este diria cual de los tres es el mayor.

Durante este periodo, tambien se hicieron practicas como figuras a las cuales porsteriormente se implemto animaciones, botones a los cuales editamos su tamaño, como la posición en la que se encuetra, todo esto con Grid con el cual tambien se hacen columnas.
