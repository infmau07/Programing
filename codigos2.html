Sistemas gestores de base de datos para Android

Se investigo sobre las distintas bases de datos para Android. las cuales fueron MySQL, SQLite, ROOM, ORM

MySQL

-Es un sistema de gestión de bases de datos relacionales (RDBMS) de código abierto que se utiliza para almacenar y gestionar datos.
-A diferencia de SQLite y Room, MySQL se ejecuta en un servidor y requiere una conexión de red para acceder a los datos.
-Puede ser una opción para aplicaciones Android que necesiten acceder a una base de datos centralizada o compartida.

SQLite

-Es una base de datos ligera y de código abierto, ideal para aplicaciones que necesitan almacenar datos localmente en el dispositivo.
-No requiere un servidor dedicado, lo que la hace fácil de integrar en aplicaciones Android.
-La propia plataforma Android proporciona herramientas para crear y gestionar bases de datos SQLite.

ROOM

-Es una biblioteca de persistencia de Android que actúa como una capa de abstracción sobre SQLite.
-Facilita la interacción con la base de datos al permitir el mapeo de objetos Java a registros de la base de datos.
-Ofrece validación de consultas en tiempo de compilación, lo que ayuda a detectar errores antes de la ejecución.
-Se integra bien con otras bibliotecas de Android como LiveData y RxJava

ORM

Es una técnica que permite mapear objetos de una aplicación a tablas de una base de datos y viceversa, simplificando el acceso y la manipulación de datos

Seguridad y niveles de seguridad para Android

La seguridad integrada de tu dispositivo siempre está activa. La inteligencia avanzada y el aprendizaje automático ayudan a protegerte de forma proactiva contra nuevas amenazas todos los días. Además, ayudan a defender tus datos contra apps malintencionadas, software malicioso, phishing y spam. Administra con comodidad la forma en que se comparten y usan tus datos. Accede a todas las opciones de tu configuración de privacidad en un solo lugar y cambia los permisos fácilmente en cualquier momento.

Niveles de seguridad

-Hardware: Android utiliza hardware específico para mejorar la seguridad, como el uso de un Secure Element para almacenar datos sensibles.
-Sistema Operativo:Android proporciona una base segura con funciones como la encriptación, la verificación de apps y la zona de pruebas para apps.
-Red: Android ofrece protección contra spam y phishing, y también alerta sobre sitios web potencialmente peligrosos.
-Seguridad Proactiva: Android ofrece funciones de seguridad proactivas que ayudan a proteger tus datos, como la protección contra software malicioso, la verificación de apps, y las actualizaciones de seguridad.
-Actualizaciones de seguridad: Proporcionan soluciones a vulnerabilidades y mejoran las funciones de seguridad.
-Encriptación: Cifra los datos almacenados en el dispositivo para proteger la información confidencial.
-Protección contra spam y phishing: Filtra los mensajes de spam y phishing, y alerta sobre sitios web peligrosos.
-Gestión de permisos: Permite a los usuarios controlar los permisos de las aplicaciones y cambiar los permisos fácilmente en cualquier momento.

Mecanismo de perfección en Android

Para garantizar el funcionamiento y correcto desempeño de cualquier dispositivo con Android se establececieron mecanismos de protección en diversos niveles.

a) Acceso o permiso a la galeria de fotos y videos.

b) Configuración de dispositivos de hardware como camara frontal y principal.

c) Acceso a la señal Wi-Fi o internet.

d) Acceso y permisos de notificación al sistema.

e) Configuración de resolución de pantalla, brillo y modo de lectura.

Por lo tanto también se considera en la forma en como se accede a travez de datos biometricos, dactilares, por contraseña o PIN.
