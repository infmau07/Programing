Conclusión

Para esta tercer unidad, se realizaron diversas actividades prácticas que formaron parte esencial de esta investigación. Las investigaciones llevadas a cabo se enfocaron principalmente en comprender las distintas bases de datos, en particular SQLite, y cómo es posible implementar dicha tecnología mediante la programación en código.

Para lograr este objetivo, se incluyeron estudios sobre la herramienta Movil.NET, así como sobre los tipos de seguridad que existen en el sistema operativo Android, abordando los distintos niveles y métodos de protección que este ofrece.

En la parte práctica del proyecto, se llevaron a cabo varios ejercicios, entre ellos la creación de una base de datos y la elaboración de un código específico que nos permitiera instalarla correctamente y compartir información entre diferentes dispositivos.

Mientras realizábamos estas prácticas, también se exploró cómo influye la seguridad en la correcta vinculación de los datos.

Finalmente, se desarrolló un proyecto con nombre y categoría propios, aplicando los conocimientos adquiridos durante todo el proceso de investigación y práctica en el área de seguridad de la computación.
