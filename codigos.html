Investigaciones

Desarrollo de aplicaciones con tecnologías multiplataforma.

Esta Investigación nos llevo a conocer sobre las distintas tecnologías multiplataforma, en la cual hay dos formas tipicas de desarrollar aplicaciones móviles, una es usando tecnologías nativas y la otra es usando tecnologías multiplataforma o tambien llamadas, Cross-Platform.
Por una parte de las tecnologías nativas es crear codigos que funciona unicamente para una plataforma.
Por otra parte las tecnologías multiplataforma con la cual podemos encontrar algunos frameworks o SDKS, lo que significa que podemos crear aplicaciones para muchas plataformas como Android y a la vez iOS a base de un unico codigo.

Sistemas operativos para smarthphone

Esta fue una Investigación que su principal objetivo era saber varios de los sistemas operativos para smarthphone, los cuales que fueron mencionados y encontrados fueron, Android, iOS, KaiOS, Sailfish OS, Harmuny OS, Blackberry, Symbian, LG webOS, Windows phone.

Configuración de dispositivos

Configuración de Visual Studio 2022

Esta Configuración debe estar colocada en Visual Studio Installer, cada cuadro marcarcado en las imagenes son necesarios para una buena función.

El codigo a usar en la consola es: dotnet --list-sdks

Configuración de computadora

En Configuraciónde la computadora, iremos al apartado de Para programadores, donde primeramente activaremos el apartado modo para desarrolladores, lo que nos permetira una mejor ejecución de las aplicaciones, despues de activarla ahora sera turno de detección de dispositivos el cual al conectar un dispositivo movil como el celular, la aplicación podra ejecutarse en esta.

(modelo de celular: Motorola G52)
Abriremos los ajuestes del celular y iremos al apartado de Acerca del teléfono, ahi presionaremos sobre numero de compilación, el cual nos dara el numero, en el caso de la imagen este ya esta activado.

(modelo de celular: Motorola G52)
Ya con el paso anterior, podremos ir a opciones para desarrolladores, en el cual activaremos la primera opción Usar opciones para desarrolladores, después mas abajo activaremos la opción depuración por USB, el cual nos permitira que al momento de conectar un cable USB desde la computadora al celular, podremos ejecutar el proyecto sin problemas.
